# DA_first_project
Organizational tool for assigning papers to reviewers in a scientific committee based on expertise.

## Linux / MacOS build instructions
````
# -S: uses the current directory as the source directory
# -B: puts all the files on a folder called build
cmake -S . -B build
# calls the compiler to build the desired targets 
cmake --build build
# To run the tool
./main
# TO run the tests
./OrgToolTests
````
