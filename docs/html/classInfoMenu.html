◆ display()

int InfoMenu::display ( ) const

Displays the information menu and handles user interaction for displaying different sections of the data. The user can choose to view submissions, reviewers, parameters, control settings, or run the max flow solver. The method continues to prompt the user until they choose to exit or run the solver.

Returns
Either 0, if the user chooses to exit, or 1, if the user chooses to run the max flow solver.
